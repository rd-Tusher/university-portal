A modern and intuitive university management portal designed to streamline and simplify academic tasks for both students and faculty. This project enables users to efficiently manage schedules, assignments, class materials, and announcements in a centralized platform.

This project is being developed as an academic initiative by **Tusher Debnath (rdTusher)** of **Mawlana Bhasani Science and Technology University**, under the supervision of **Professor Dr. Mahbuba Begum**.
